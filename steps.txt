# Push to GitHub

- If it's your first time pushing, set the remote and branch
git remote add origin <your-repo-url>
git branch -M main
git push -u origin main

- If your repository is linked to GitHub, push the changes:
git remote add origin <repo_url>   # Add a remote repository
git remote -v   # List remote repositories
git push -u origin <branch_name>   # Push changes to remote branch
git pull origin <branch_name>   # Pull latest changes from remote
git fetch origin   # Fetch changes without merging
